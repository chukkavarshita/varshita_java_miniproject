# Java Quiz Game

## Overview

Java Quiz Game is a console-based mini project developed using Java. The application allows users to select a difficulty level (Easy, Medium, or Hard), answer multiple-choice questions, receive instant feedback, and view their final score along with a performance evaluation.

## Features

- Easy, Medium, and Hard difficulty levels
- Multiple-choice questions
- Instant answer validation
- Automatic score calculation
- Performance-based feedback
- Simple console-based interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Arrays
- Loops
- Conditional Statements

## How to Run

1. Compile the program:
   ```
   javac Main.java
   ```
2. Run the program:
   ```
   java Main
   ```
